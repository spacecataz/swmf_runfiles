The files in this directory are for runs done in collaboration
with Weichao Tu's group for the purpose of specifying magnetopause
location during events of interest to her group.

The solar wind has been taken from the Wind satellite for the 3/11/16
event, a minor storm with a period of high densities.

The initial run for this study is a SWPC v2 style standard run.
Run `make test_swpc` to configure the SWMF for this run.
