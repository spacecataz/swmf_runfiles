In this directory are input files for the January 19, 2026
geomagnetic storm. `IMF_wind_unprop.dat` contains unpropagated
Wind K0 level data that has been filtered and resampled to 1 min.
`IMF_wind_bflip.dat` contains the same Wind data but with the IMF
components reversed, for the sake of experiment: would this storm
have cause a 1/100 year GMD event if Bz had pointed southward
instead of northward?
