# Auroral Investigations

This folder is for simulations whose goal is to investigate auroral
activity.

## Description of Subfolders

| Subfolder | Description |
|-----------|-------------|
|`SMC_1998` | Inputs for recreating aurora during the Sept. 23, 1998 SMC event. |
|`PMAFs`    | Simulations pertaining to Poleward Moving Auroral Forms |
